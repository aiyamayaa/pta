#14. 最长公共前缀

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。



示例 1：
```go
输入：strs = ["flower","flow","flight"]
输出："fl"
```

示例 2：
```go
输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。
```



提示：
```go
1 <= strs.length <= 200
0 <= strs[i].length <= 200
strs[i] 仅由小写英文字母组成
```

##思路：
使用逐渐截取删除的方法，首先令返回的值`res = strs[0]`，然后依次遍历所有的子字符串，每次都截取字符子串和res相同的部分，遍历结束返回最终的结果.(纵向扫描)
