# Pow(x, n)
实现pow(x, n)，即计算 x 的整数n 次幂函数（即，xn ）。


示例 1：
```azure
输入：x = 2.00000, n = 10
输出：1024.00000
```

示例 2：
```azure
输入：x = 2.10000, n = 3
输出：9.26100
```

示例 3：
```azure
输入：x = 2.00000, n = -2
输出：0.25000
解释：2-2 = 1/22 = 1/4 = 0.25
```



提示：
```azure
-100.0 < x < 100.0
-231 <= n <= 231-1
-104 <= xn <= 104
```

##思路
使用快速幂 + 递归
即：x的n次幂=x的(n/2)次幂*x的(n/2)次幂
- 如果(n%2)==1，需要再乘一下x
- 否则直接返回即可
